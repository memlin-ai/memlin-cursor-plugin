Memlin features (a.k.a. Workstreams) — project-scoped units of work that gather the thoughts, plans, goals, and PRs behind one thing. Run `memlin features list` to see them, `memlin features create "<title>"` to make one in the bound project, or `memlin features add <feature-id> <kind> <id>` to tag an existing thought/memory/plan/goal/todo/decision/schema/file into a feature, all from the terminal.
